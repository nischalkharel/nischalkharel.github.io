RemoteCLIP Video Retrieval Pipeline

Graduate Research — NSF SHREC Center

High-performance text-based video retrieval system for satellite and remote sensing footage. Built a parallel C++ embedding pipeline using LibTorch and OpenMP with a producer-consumer architecture, and benchmarked brute-force kNN retrieval across serial CPU, parallel CPU (OpenMP), and GPU (cuBLAS) backends. Key findings include NUMA-aware scaling achieving ~38x speedup, GPU warm search ~185x faster than serial, and GPU cold-start end-to-end slower than CPU serial due to CUDA init and PCIe overhead. GSI Gemini II APU benchmarking in progress.

Class-Adaptive Satellite Image Compression on FPGA

Graduate Research — NSF SHREC Center · Targeting SCC Conference

Developing a class-adaptive and confidence-adaptive compression pipeline for satellite imagery on the Xilinx ZCU102 FPGA. The system tiles satellite images, classifies each tile by land-cover type using a CNN (NWPU-RESISC7), then assigns per-tile CNNJPEG quality factors based on each class's compression tolerance rather than using a single fixed quality for the entire image. The approach aims to save significant bandwidth while maintaining classification stability, with a reusable five-step calibration methodology. Work in progress.

Fault Tolerance for GPU-Accelerated Neural Networks in Space

Graduate Research — NSF SHREC Center

Evaluated software fault tolerance techniques — Triple Modular Redundancy (TMR), Dual+Retry, and Algorithm-Based Fault Tolerance (ABFT) — for neural networks on NVIDIA Jetson hardware targeting space radiation environments. Used PyTorchFI for fault injection after NVBitFI proved incompatible with SM_87 architecture. Key finding: Dual+Retry achieved 0% silent data corruption with 34% better efficiency than TMR.

CNNJPEG on LM LINUSS CubeSat

NSF SHREC Center × Lockheed Martin

Optimized a CNN-based image compression model (CNNJPEG) for the Xilinx ZCU102 FPGA under severe memory constraints, implementing dynamic tiling and quantization to maintain 20x compression while reducing memory usage by 70%. Collaborated with Lockheed Martin to deploy the application to the LM LINUSS CubeSat in geosynchronous orbit, 22,236 miles above Earth. As of 2026, the application has been successfully tested on the actual LINUSS satellite in orbit.
