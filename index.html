RemoteCLIP Video Retrieval Pipeline

Graduate Research — NSF SHREC Center

High-performance text-based video retrieval system for satellite and remote sensing footage. Built a parallel C++ embedding pipeline using LibTorch and OpenMP with a producer-consumer architecture, and benchmarked brute-force kNN retrieval across serial CPU, parallel CPU (OpenMP), and GPU (cuBLAS) backends. Key findings include NUMA-aware scaling achieving ~38× speedup, GPU warm search ~185× faster than serial, and a publishable result showing GPU cold-start end-to-end is slower than CPU serial due to CUDA initialization and PCIe overhead — significant for latency-sensitive edge and space deployments. GSI Gemini II APU benchmarking is currently in progress.

Fall 2025

Fault Tolerance for GPU-Accelerated Neural Networks in Space

Graduate Research — NSF SHREC Center

Evaluated software fault tolerance techniques — Triple Modular Redundancy (TMR), Dual+Retry, and Algorithm-Based Fault Tolerance (ABFT) — for neural networks on NVIDIA Jetson hardware targeting space radiation environments. Used PyTorchFI for fault injection after NVBitFI proved incompatible with SM_87 architecture. Key finding: Dual+Retry achieved 0% silent data corruption with 34% better efficiency than TMR. Co-authored a paper on the results.

Spring 2025 — 2026

CNNJPEG on LM LINUSS CubeSat

NSF SHREC Center × Lockheed Martin

Optimized a CNN-based image compression model (CNNJPEG) for the Xilinx ZCU102 FPGA under severe memory constraints, implementing dynamic tiling and quantization to maintain 20× compression while reducing memory usage by 70%. Collaborated with Lockheed Martin to package and deploy the application to the LM LINUSS CubeSat in geosynchronous orbit, 22,236 miles above Earth. As of 2026, the application has been successfully tested on the actual LINUSS satellite in orbit.
